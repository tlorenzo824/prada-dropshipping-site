About Prada

Prada is a student-built dropshipping concept store focused on modern, affordable room upgrades. The mission is simple: help everyday dreamers build a space that feels clean, motivating, and stylish.

This site demonstrates responsive design, accessibility basics, and secure input handling (client-side) for a course final project.

Accessibility + Ethics

  • Images include descriptive alt text
  • Keyboard-friendly navigation (skip link)
  • Clear contrast and readable typography
  • No sensitive personal data is stored
